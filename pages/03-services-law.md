# 3 障害者総合支援法と児童福祉法のサービス

障害のある方が地域で暮らし続けられるよう、**障害者総合支援法**と**児童福祉法**に基づくサービスを提供。

## (1) 制度の全体像

- **障害者総合支援法** = 自立支援給付 ＋ 地域生活支援事業（区市町村が地域特性に応じ柔軟に実施）。
- **児童福祉法** = 障害児通所支援・障害児入所支援・障害児相談支援。

```
自立支援給付
 ├ 介護給付: 居宅介護(ホームヘルプ)/重度訪問介護/行動援護/同行援護/短期入所(ショートステイ)/
 │           重度障害者等包括支援/療養介護/生活介護/施設入所支援
 ├ 訓練等給付: 自立訓練/宿泊型自立訓練/就労移行支援/就労選択支援(R7.10.1施行)/就労継続支援(A/B型)/
 │           就労定着支援/自立生活援助/共同生活援助(グループホーム)
 ├ 自立支援医療: 更生医療/育成医療/精神通院医療 → [[05-medical]]
 ├ 補装具 → [[06-assistive-devices]]
 ├ 相談支援: 計画相談支援/地域相談支援
地域生活支援事業: 意思疎通支援 → [[09-information-support]] / 日常生活用具 → [[06-assistive-devices]] /
                移動支援 → [[07-daily-life]] / 成年後見制度利用支援 → [[19-rights-advocacy]] /
                地域活動支援センター / 日中短期入所 など
児童福祉法: 児童発達支援/放課後等デイサービス/居宅訪問型児童発達支援/保育所等訪問支援/
           障害児相談支援/福祉型・医療型障害児入所施設 → [[14-children-services]]
```

## (2) 主なサービスの種類・内容

### 介護給付
- **居宅介護（ホームヘルプ）**: 居宅での入浴・排せつ・食事の介護、家事援助、通院介助。
- **重度訪問介護**: 重度肢体不自由者/重度知的・精神障害で行動上著しい困難があり常時介護が必要な方に、居宅介護・家事・外出時の移動支援等を総合的に。
- **同行援護**: 視覚障害のある方の外出時の移動援護。
- **行動援護**: 知的/精神障害で行動上著しい困難があり常時介護が必要な方の危険回避・外出介護。
- **重度障害者等包括支援**: 介護の必要性が非常に高い方に障害福祉サービスを包括的に。
- **短期入所（ショートステイ）**: 短期間入所し夜間含め介護・支援。
- **療養介護**: 医療と常時介護が必要な方に医療機関で機能訓練・看護・介護等。
- **生活介護**: 常時介護が必要な方に主に昼間、施設で介護・支援・創作的活動等。
- **施設入所支援**: 施設入所者への夜間等の介護。

### 訓練等給付
- **自立訓練（機能訓練・生活訓練）**: 一定期間、身体機能/生活能力向上の訓練。
- **宿泊型自立訓練**: 地域移行に向け居室等を利用し日常生活能力を向上。
- **就労移行支援**: 一般就労希望者に就労に必要な知識・能力向上の訓練。
- **就労選択支援**（令和7年10月1日施行）: 就労アセスメントで本人に合った選択を支援。
- **就労継続支援（A型・B型）**: 一般就労が困難な方に働く場と訓練。
- **就労定着支援**: 一般就労した方の生活課題への支援。
- **自立生活援助**: 入所/GHから居宅へ移行した方への巡回訪問等の援助。
- **共同生活援助（グループホーム）**: 共同生活住居での介護・相談・移行支援。→ 家賃助成 [[04-allowances-pensions]]

### 相談支援
- **地域相談支援**（地域移行支援・地域定着支援）: 施設/病院からの地域移行の相談・住居確保・連絡サポート。
- **計画相談支援**（サービス利用支援・継続サービス利用支援）: サービス等利用計画の作成。

### 地域生活支援事業（章横断リンク）
理解促進研修・啓発／自発的活動支援／相談支援事業／意思疎通支援（→ [[09-information-support]]）／日常生活用具給付（→ [[06-assistive-devices]]）／移動支援（→ [[07-daily-life]]）／成年後見制度利用支援（→ [[19-rights-advocacy]]）／地域活動支援センター／日中短期入所／巡回入浴（→ [[07-daily-life]]）／救急代理通報（→ [[08-disaster-emergency]]）／自動車運転免許取得・改造費補助（→ [[10-taxi-car-parking]]）／重度障害者等就労支援（→ [[07-daily-life]]）

### 児童福祉法サービス（→詳細 [[14-children-services]]）
児童発達支援／放課後等デイサービス／居宅訪問型児童発達支援／保育所等訪問支援／障害児相談支援事業／福祉型・医療型障害児入所施設。

## 申請から利用までの流れ
①相談・申請（区窓口・指定相談支援事業所）→ ②認定調査（80項目の全国共通調査＋医師意見書。介護給付は一次判定〈コンピュータ〉→二次判定〈審査会で区分1〜6認定〉。※18歳未満の障害児は認定調査なし）→ ③サービス利用意向聴取・利用計画案提出（指定相談支援事業者作成 or セルフプラン可）→ ④支給決定 → ⑤サービス等利用計画書作成 → ⑥利用契約・利用開始。

**問い合わせ**: 身体=障害福祉課身体障害者支援係☎03-5803-1219／知的=知的障害者支援係☎1214／精神・難病=予防対策課精神保健担当☎03-5803-1847（難病対象疾患→ [[99-reference]] 原本199p）

## (3) 利用者負担
所得に応じた月額上限が設定され、上限を超える負担は生じない。

### 月ごとの利用者負担上限額
| 区分 | 世帯の収入状況 | 上限月額 |
|---|---|---|
| 生活保護 | 生活保護受給世帯 | 0円 |
| 低所得1 | 住民税非課税世帯で本人年収80万円以下 | 0円 |
| 低所得2 | 住民税非課税世帯（低所得1除く） | 0円 |
| 一般1 | 住民税課税・所得割16万円未満（障害児・20歳未満施設入所者は28万円未満） | 障害児:4,600円／障害者・20歳未満施設入所者:9,300円 |
| 一般2 | 住民税課税世帯（一般1除く） | 37,200円 |

※補装具は課税世帯だと「一般2」のみ（一定所得以上は対象外）。入所施設(20歳以上)・GH利用の課税世帯は「一般2」。
※所得判断の世帯範囲: 18歳以上の障害者=本人と配偶者／障害児=保護者の属する住民票世帯。

### 国の軽減制度
①医療型個別減免 ②食費等実費負担の軽減（通所の食費、入所の食費・光熱水費、GH家賃は月1万円上限で補足給付）③就学前障害児通所の多子軽減 ④幼児教育・保育無償化（3〜5歳、手続き不要）⑤高額障害福祉サービス等給付費（併給・世帯合算が基準超で償還）＋新高額（65歳到達で介護保険移行時の負担軽減）⑥生活保護への移行防止策。

### 都・区の軽減制度
- **都**: 児童発達支援事業所等利用支援事業（0〜2歳の第2子以降無償化。文京区は区独自制度で手続き不要）。
- **区**: ①利用者負担を一体管理し上限超えを防止 ②就学前障害児通所の利用料・食費を全額助成（要件あり）③同行援護・移動支援は月36時間まで負担なし ④意思疎通支援は負担なし ⑤日常生活用具（ストマ用装具・紙おむつ）は基準額内負担なし ⑥文京区グループホーム家賃助成（生活保護非受給者、所得に応じ一部）⑦給食費の軽減（大塚・小石川福祉作業所/本郷福祉センター=非課税230円・課税350円等、教育センターそよかぜ=非課税110円等）。
