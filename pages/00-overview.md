# 概要・利用にあたって・シンボルマーク

## このてびきについて
文京区『令和7年 障害者福祉のてびき』は、障害のある方に対するサービスの内容と利用方法をまとめた冊子。
- 内容基準日: **令和7年5月末現在**（発行後に金額等が変わりうる）。Web版は「文京区 てびき」で検索。UDフォント採用。
- 対象記号: **身**=身体／**知**=知的／**精**=精神／**難**=難病／**児**=18歳未満の障害児。→ [[target-categories]]
- ソースの詳細 → [[r7-tebiki]]（sources/）

## 障害福祉に関するシンボルマーク（原本 表紙裏）
てびき掲載の主なマークと問い合わせ先:

| マーク | 意味 | 主な団体 |
|---|---|---|
| 障害者のための国際シンボルマーク | 障害のあるすべての人が利用しやすい建物・設備等を示す世界共通マーク | 日本障害者リハビリテーション協会 |
| 盲人のための国際シンボルマーク | 視覚障害者への配慮がなされた設備等の世界共通マーク | 日本盲人福祉委員会 |
| 耳マーク | 聞こえが不自由なことを示す／聴覚障害者への配慮 | 全日本難聴者・中途失聴者団体連合会 |
| ヒアリングループマーク | Tコイルを使って利用できる施設・機器 | 全日本難聴者・中途失聴者団体連合会 |
| オストメイトマーク | 人工肛門・人工膀胱を使用する人／オストメイト対応設備 | 交通エコロジー・モビリティ財団 |
| ハート・プラスマーク | 身体内部（心臓・呼吸機能等）に障害のある人 | ハート・プラスの会 |
| ほじょ犬マーク | 身体障害者補助犬（盲導犬・介助犬・聴導犬）の啓発 | 東京都福祉局 |
| ヘルプマーク | 援助や配慮を必要とすることが外見からわからない人 | 東京都福祉局 → [[08-disaster-emergency]] |
| 身体障害者標識／聴覚障害者標識 | 運転者が身体/聴覚障害であることを示す | 警視庁交通局 |
| 白杖SOSシグナル | 白杖を頭上に掲げSOSを示す視覚障害者への支援啓発 | 岐阜市福祉部 |
| 手話マーク／筆談マーク | 手話／筆談での対応ができることを示す | 全日本ろうあ連盟 |
| すけだちくん | 共生社会実現に向けた障害者理解促進キャラクター | ― |
