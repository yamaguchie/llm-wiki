# 参考資料（等級表・疾病一覧・所得制限）

各制度の判定基準・対象疾病・所得制限をまとめた巻末資料。手帳の概要は [[disability-notebooks]]。

## 1. 身体障害者障害程度等級表（原本191-194p）
身体障害者福祉法施行規則別表第5号。障害の種類（視覚／聴覚又は平衡機能／音声・言語・そしゃく機能／肢体不自由〈上肢・下肢・体幹・乳幼児期以前の非進行性脳病変による運動機能〉／内部障害〈心臓・じん臓・呼吸器・ぼうこう又は直腸・小腸・HIVによる免疫・肝臓〉）ごとに1〜6級を規定。
- 備考: 二つの重複障害は原則一級上位。肢体不自由で7級が2つ以上重複する場合は6級。異なる等級の重複は障害程度を勘案して上位等級とできる。
- 詳細な級別の状態記述は原本を参照（抽出テキストでは表が判読困難）。

## 2. 知的障害（愛の手帳）判定基準表（原本195-197p）
就学前(0〜6歳)・児童(6〜17歳)・成人(18歳以上)の3表。知能指数(IQ)の目安と、運動機能・社会性・意思疎通・身体的健康・基本的生活等を総合判定。
| 度 | 区分 | IQの目安 |
|---|---|---|
| 1度 | 最重度 | おおむね19以下 |
| 2度 | 重度 | おおむね20〜34 |
| 3度 | 中度 | おおむね35〜49 |
| 4度 | 軽度 | おおむね50〜75 |

## 3. 精神障害者保健福祉手帳障害等級判定基準（原本198p）
精神疾患（機能障害）の状態と能力障害（活動制限）の状態の両面から判定。
- **1級**: 日常生活の用を弁ずることを不能ならしめる程度（食事・清潔保持・金銭管理・通院服薬・意思伝達等が不能）。
- **2級**: 日常生活が著しい制限を受ける程度（上記が援助なしにはできない）。
- **3級**: 日常生活・社会生活が制限を受ける程度（おおむねできるがなお援助を必要とする）。

## 4. 障害者総合支援法の難病患者等対象疾病一覧（原本199-201p）
**376疾病**（令和7年4月1日改定）。うち○印は障害者総合支援法独自の支援疾病（29疾病）。この対象疾病に該当する難病患者等は、手帳がなくても障害福祉サービス・補装具・日常生活用具等の対象となる（→ [[03-services-law]] / [[06-assistive-devices]]）。
- 全疾病名は原本に五十音・アルファベット順で掲載（アイカルディ症候群〜肋骨異常を伴う先天性側弯症、ATR-X症候群等）。

## 5. 心身障害者等福祉手当（区制度）指定の疾病一覧（原本201-203p）
**361疾病**（令和7年4月1日改定）。この指定疾病にかかり医療券等を所持する方は心身障害者等福祉手当の対象（→ [[04-allowances-pensions]]）。全疾病名は原本に掲載。

## 6. 所得制限限度額基準表（原本204-205p）
所得＝年間総収入額（税込）から給与所得控除（給与所得）又は必要経費（事業所得等）を差し引いた額。他に各種控除あり。※年度途中で変更の可能性あり。
