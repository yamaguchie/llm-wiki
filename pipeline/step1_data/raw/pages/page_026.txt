# (15) 生活あんしん拠点（地域生活支援拠点）

障害者とその家族が安心して地域に住み続けることができるように、障害者等からの相談支援や地域づくりなどを行っています。

| | |
| --- | --- |
| **対 象** | 区内在住の障害者及びその家族や関係機関の方 |
| **開所時間** | 【月〜金曜日】午前10時〜午後5時30分<br>※土・日・祝日・年末年始は休み |
