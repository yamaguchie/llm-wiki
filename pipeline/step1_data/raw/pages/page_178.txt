### （１）教育センター（児童発達支援センター） [身][知][精][難][児]

教育センター（児童発達支援センター）では、身近な地域における通所支援機能である児童発達支援や放課後等デイサービス、障害児相談支援などの地域支援に対応し、切れ目のないサービスの提供を行います。

#### 【児童発達支援「そよかぜ」】

| 項目 | 内容 |
| --- | --- |
| **内 容** | 心身の発達に何らかの遅れや偏りのある就学前の乳幼児を対象に、日常生活における基本動作の獲得や集団生活などへの適応力の向上を図ります。 |
| **利用日・時間** | 【月〜金曜日】 午前9時30分〜午後2時<br>【第2・4土曜日】 午前9時30分〜午前11時30分 |

#### 【放課後等デイサービス「ほっこり」】

| 項目 | 内容 |
| --- | --- |
| **内 容** | 心身の発達に何らかの遅れや偏りのある就学児（小学生）を対象に、社会生活・集団生活などへの適応力の向上を図ります。 |
| **利用日・時間** | 【月〜金曜日】 午後2時〜午後6時<br>【第2・4土曜日及び長期休み】 午後1時〜午後5時 |

#### 【特定相談支援事業及び障害児相談支援事業「なないろ」】

| 項目 | 内容 |
| --- | --- |
| **内 容** | お子さんが児童発達支援や放課後等デイサービスなどの福祉サービスを利用する際に必要な「サービス等利用計画案」を相談支援専門員が作成します。給付決定後は、モニタリングにより計画の実施状況の把握等を行い、必要に応じて計画の変更を行うほか、お子さんが利用するサービス事業者等との連絡調整を行います。 |
| **利用日・時間** | 【月〜金曜日及び第2・4土曜日】 午前9時〜午後5時 |
| **休 館 日** | 日曜日・祝日、12月29日〜1月3日 |
| **問い合わせ先** | 〒113-0034 文京区湯島4-7-10 FAX 03-5800-2590<br>〔児童発達支援「そよかぜ」〕 TEL 03-5800-2631<br>〔放課後等デイサービス「ほっこり」〕 TEL 03-5800-2632<br>〔特定相談支援事業及び障害児相談支援事業「なないろ」〕<br>TEL 03-5800-2594<br>（地図は、19ページに掲載しています。） |
