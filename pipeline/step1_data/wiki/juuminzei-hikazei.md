# 住民税の非課税

年間の合計所得金額が一定以下の障害者に対して、翌年の住民税を非課税とする制度です。

## 概要

年間の合計所得金額が135万円以下の障害者は、翌年の住民税が課税されません。

## 対象となる方

以下のいずれかに該当する方
1. [身体障害者手帳](shintai-shougaisha-techou.md) 1〜6級を所持する方
2. [愛の手帳](ai-no-techou.md) 1〜4度を所持する方
3. [精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md) 1〜3級を所持する方
4. 精神上の障害により事理を弁識する能力を欠く常況にある方等
5. 精神又は身体に障害のある65歳以上の方で、1から4に準ずるものとして市区町村長等に認定を受けている方
