# 裁判員制度に参加する障害者への支援

裁判員制度に参加する障害者への支援は、障害のある方が裁判員、補充裁判員、または裁判員候補者に選任された際、裁判手続に円滑に参加できるよう、最高裁判所および地方裁判所が必要な配慮やサポートを提供する制度です。

## 主な支援・配慮の内容
個々の障害特性や希望に基づき、以下のような配慮がなされます。

- **聴覚障害・音声機能障害への配慮**:
  - [手話通訳者の派遣・要約筆記者の派遣](shuwa-yakuji-haken.md)（手話通訳人・要約筆記人の手配）
  - 磁気ループ（ヒアリングループ）や、音声認識ソフトによる文字表示機器の提供
- **視覚障害への配慮**:
  - 裁判手続に関連する資料（起訴状など）の点字化・音声デイジー化
  - 拡大読書器や、音声読み上げパソコンの設置
- **肢体不自由（身体障害）への配慮**:
  - 車いす対応のスロープ、多目的トイレ、車いす用の傍聴・審理席の確保
  - 補助犬（盲導犬・介助犬・聴導犬等）の同伴
- **精神障害・知的障害・発達障害への配慮**:
  - 専門用語の分かりやすい言い換え、休憩時間の頻度の調整、個別相談室の確保

## 問い合わせ
- 選任通知が届いた段階で、当該の**地方裁判所（東京地方裁判所など）**の裁判員係へ直接ご相談・ご要望をお伝えください。
- 一般的な相談窓口: [障害福祉課](shogai-fukushi-ka.md)、[文京区障害者基幹相談支援センター](bunkyo-disability-key-consultation-support-center.md)
