# 文京区手話言語条例 / 障害者による情報の取得及び利用並びに意思疎通の促進に関する条例

障害のある人もない人も共に生きるまちをつくるため、多様なコミュニケーションによる意思疎通や情報の取得を促進することを目的として、令和6年4月に施行された条例です。

## 概要と啓発の取り組み
- **制定日**: 令和6年4月
- **パンフレットの配布**:
  本条例の趣旨を広めるための周知啓発パンフレット「いろいろなコミュニケーションで、心と心がつながる文京区」を作成し、[障害福祉課](shougai-fukushi-ka.md)で配布しているほか、文京区のホームページでも公開しています。
