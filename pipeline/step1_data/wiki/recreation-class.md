# レクリエーション教室

レクリエーション教室は、障害のある方が気軽に体を動かし、レクリエーションゲームなどを通じて他者との交流を楽しむための教室です。

## 概要
運動が苦手な方でも楽しめるよう、ルールを工夫したゲームや軽スポーツを中心にプログラムが構成されています。参加者の体力に合わせて無理なく活動できることが特徴です。

## 活動内容
- レクリエーションスポーツ（風船バレー、ボッチャなど）
- リズム体操
- グループで行うゲーム
