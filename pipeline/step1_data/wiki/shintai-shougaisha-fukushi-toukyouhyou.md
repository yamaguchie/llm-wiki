# 身体障害者福祉等級表

[身体障害者福祉等級表](shintai-shougaisha-fukushi-toukyouhyou.md)は、身体障害者福祉法施行規則別表第5号に基づく、身体障害の程度を1級から7級に区分した基準表です。

## 級別基準

### 1級
- **視覚障害**: 両眼の視力（屈折異常がある場合は矯正視力）の和が0.01以下のもの
- **肢体不自由（上肢）**: 一上肢の機能を全廃したもの、または両上肢を手関節以上で欠くもの
- **肢体不自由（下肢）**: 一下肢の機能を全廃したもの、または両下肢を大腿の2分の1以上で欠くもの
- **肢体不自由（体幹）**: 体幹の機能障害により坐っていることができないもの
- **乳幼児期以前の脳病変による運動機能障害**:
  - 【上肢機能】不随意運動・失調等により上肢を使用する日常生活動作がほとんど不可能なもの
  - 【移動機能】不随意運動・失調等により歩行が不可能なもの
- **内部障害（心臓、じん臓、呼吸器、ぼうこう・直腸、小腸、ヒト免疫不全ウイルスによる免疫、肝臓）**: 各機能障害により、自己の身辺の日常生活活動（または日常生活）が極度に制限される、またはほとんど不可能なもの

### 2級
- **視覚障害**: 
  - 両眼の視力の和が0.02以上0.04以下のもの
  - 両眼の視野がそれぞれ10度以内で、かつ両眼による視野について視能率による損失率が95パーセント以上のもの
- **聴覚障害**: 両耳の聴力レベルがそれぞれ100デシベル以上のもの（両耳全ろう）
- **肢体不自由（上肢）**: 両上肢の機能の著しい障害、両上肢のすべての指を欠くもの、一上肢を上腕の2分の1以上で欠くもの、または一上肢の機能を全廃したもの
- **肢体不自由（下肢）**: 両下肢の機能の著しい障害、または両下肢を下腿の2分の1以上で欠くもの
- **肢体不自由（体幹）**: 体幹の機能障害により坐位もしくは起立位を保つことが困難、または立ち上がることが困難なもの
- **乳幼児期以前の脳病変による運動機能障害**:
  - 【上肢機能】不随意運動・失調等により上肢を使用する日常生活動作が極度に制限されるもの
  - 【移動機能】不随意運動・失調等により歩行が極度に制限されるもの

### 3級
- **視覚障害**:
  - 両眼の視力の和が0.05以上0.08以下のもの
  - 両眼の視野がそれぞれ10度以内で、かつ両眼による視野について視能率による損失率が90パーセント以上のもの
- **聴覚障害**: 両耳の聴力レベルが90デシベル以上のもの（耳介に接しなければ大声を理解し得ないもの）
- **平衡機能障害**: 平衡機能の極めて著しい障害
- **音声、言語、そしゃく機能障害**: 音声機能、言語機能またはそしゃく機能の喪失
- **肢体不自由（上肢）**: 両上肢のおや指およびひとさし指を欠く（または機能を全廃した）もの、一上肢の機能の著しい障害、または一上肢のすべての指を欠く（または機能を全廃した）もの
- **肢体不自由（下肢）**: 両下肢をショパール関節以上で欠くもの、一下肢を大腿の2分の1以上で欠くもの、または一下肢の機能を全廃したもの
- **肢体不自由（体幹）**: 体幹の機能障害により歩行が困難なもの
- **乳幼児期以前の脳病変による運動機能障害**:
  - 【上肢機能】不随意運動・失調等により上肢を使用する日常生活動作が著しく制限されるもの
  - 【移動機能】不随意運動・失調等により歩行が家庭内での日常生活活動に制限されるもの
- **内部障害**: 各機能障害により、家庭内での日常生活活動が著しく制限されるもの（ヒト免疫不全ウイルスおよび肝臓機能障害については、社会での日常生活活動が著しく制限されるものを除く）

### 4級
- **視覚障害**: 両眼の視力の和が0.09以上0.12以下のもの、または両眼の視野がそれぞれ10度以内のもの
- **聴覚障害**: 両耳の聴力レベルがそれぞれ80デシベル以上のもの、または両耳による普通話声の最良の語音明瞭度が50パーセント以下のもの
- **音声、言語、そしゃく機能障害**: 音声機能、言語機能またはそしゃく機能の著しい障害
- **肢体不自由（上肢）**: 一上肢のおや指を欠く（または機能を全廃した）もの、一上肢の肩・肘・手関節のうちいずれか一関節の機能を全廃したもの、一上肢のおや指およびひとさし指を欠く（または機能を全廃した）もの、おや指またはひとさし指を含めて一上肢の三指を欠く（または機能を全廃した）もの、またはおや指またはひとさし指を含めて一上肢の四指の機能の著しい障害
- **肢体不自由（下肢）**: 同下肢のすべての指を欠く（または機能を全廃した）もの、一下肢を下腿の2分の1以上で欠くもの、一下肢の機能の著しい障害、一下肢の股関節または膝関節の機能を全廃したもの、あるいは一下肢が健側に比して10センチメートル以上（または健側の長さの10分の1以上）短いもの
- **乳幼児期以前の脳病変による運動機能障害**: 不随意運動・失調等により、上肢機能または移動機能において社会での日常生活活動が著しく制限されるもの
- **内部障害**: 各機能障害により、社会での日常生活活動が著しく制限されるもの

### 5級
- **視覚障害**: 両眼の視力の和が0.13以上0.2以下のもの、または両眼による視野の2分の1以上が欠けているもの
- **平衡機能障害**: 平衡機能の著しい障害
- **肢体不自由（上肢）**: 一上肢のおや指の機能の著しい障害、一上肢の肩・肘・手関節のうちいずれか一関節の機能の著しい障害（いずれも障害）、一上肢のおや指を欠く（または機能を全廃した）もの、一上肢のおや指およびひとさし指の機能の著しい障害、またはおや指またはひとさし指を含めて一上肢の三指の機能の著しい障害
- **肢体不自由（下肢）**: 一下肢の股関節または膝関節の機能の著しい障害、一下肢の足関節の機能を全廃したもの、あるいは一下肢が健側に比して5センチメートル以上（または健側の長さの15分の1以上）短いもの
- **肢体不自由（体幹）**: 体幹の機能の著しい障害
- **乳幼児期以前の脳病変による運動機能障害**: 不随意運動・失調等により、上肢機能または移動機能において社会活動に支障のあるもの

### 6級
- **視覚障害**: 一眼の視力が0.02以下、他眼の視力が0.6以下のもので、両眼の視力の和が0.2を超えるもの
- **聴覚障害**: 両耳の聴力レベルが70デシベル以上のもの（40センチメートル以上の距離で発声された会話語を理解し得ないもの）、または一側耳の聴力レベルが90デシベル以上で他側耳の聴力レベルが50デシベル以上のもの
- **肢体不自由（上肢）**: 一上肢のおや指の機能の著しい障害、ひとさし指を含めて一上肢の二指を欠く（または機能を全廃した）もの
- **肢体不自由（下肢）**: 一下肢をリスフラン関節以上で欠くもの、または一下肢の足関節の機能の著しい障害
- **乳幼児期以前の脳病変による運動機能障害**: 不随意運動・失調等により、上肢機能または移動機能が劣るもの

### 7級
- **肢体不自由（上肢）**: 一上肢の機能の軽度の障害、一上肢の肩・肘・手関節のうちいずれか一関節の機能の軽度の障害、一上肢の手指の機能の軽度の障害、ひとさし指を含めて一上肢の二指の機能の著しい障害、一上肢のなか指・くすり指・小指を欠く（または機能を全廃した）もの
- **肢体不自由（下肢）**: 両下肢のすべての指の機能の著しい障害、一下肢の機能の軽度の障害、一下肢の股・膝・足関節のうちいずれか一関節の機能の軽度の障害、一下肢のすべての指を欠く（または機能を全廃した）もの、あるいは一下肢が健側に比して3センチメートル以上（または健側の長さの20分の1以上）短いもの
- **乳幼児期以前の脳病変による運動機能障害**: 上肢または下肢に不随意運動・失調等を有するもの

## 備考
1. 同一の等級について二つの重複する障害がある場合は、一級上の級とします。ただし、二つの重複する障害が特に本表中に指定されているものは、その該当等級となります。
2. 肢体不自由において、7級に該当する障害が2つ以上重複する場合は、6級となります。
3. 異なる等級について二つ以上の重複する障害がある場合は、障害の程度を勘案して当該等級より上位の等級とすることができます。
4. 「指を欠くもの」とは、おや指については指骨間関節、その他の指については第一指骨間関節以上を欠くものをいいます。
5. 「指の機能障害」とは、中手指節関節以下の障害をいい、おや指については対向運動障害も含みます。
6. 上肢または下肢の短縮の長さは、実用長（上肢は肩峰より、下肢は大腿の坐骨結節の高さより計測したもの）をもって計測します。
7. 下肢の長さは、前腸骨棘より内くるぶし下端までを計測します。
