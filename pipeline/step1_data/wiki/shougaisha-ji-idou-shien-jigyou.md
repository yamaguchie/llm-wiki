# 障害者・児移動支援事業

屋外での移動が困難な障害者（児）の円滑な外出を支援するガイドヘルパー等の派遣事業です。

## 申請窓口

対象となる障害種別に応じて、窓口が分かれています。

- **身体障害のある方・児童**:
  - [障害福祉課](shougai-fukushi-ka.md) 身体障害者支援係（文京シビックセンター9階北側 / TEL: 03-5803-1219 / FAX: 03-5803-1352）
- **知的障害のある方・児童**:
  - [障害福祉課](shougai-fukushi-ka.md) 知的障害者支援係（文京シビックセンター9階北側 / TEL: 03-5803-1214 / FAX: 03-5803-1352）
- **精神障害（発達障害含む）のある方・難病患者**:
  - 予防対策課精神保健担当（文京シビックセンター8階南側 / TEL: 03-5803-1847 / FAX: 03-5803-1355）
