# 権利擁護センター「あんしんサポート文京」

権利擁護センター「あんしんサポート文京」は、文京区社会福祉協議会が運営する相談・支援機関です。障害者（[愛の手帳](ai-no-techou.md)所持者、精神障害のある方、身体障害のある方など）が、住み慣れた地域社会で安心して暮らし続けられるよう、権利擁護に関する各種支援事業や専門相談を実施しています。

## 実施している主な支援・事業

| 支援の種類 | 支援の内容 |
| --- | --- |
| **[福祉サービス利用援助事業（地域福祉権利擁護事業）](welfare-service-assistance.md)** | 知的障害者・精神障害者等で判断能力が十分でない方への福祉サービス利用援助、日常的金銭管理、書類預かり等の支援。 |
| **[財産保全管理サービス](asset-preservation-service.md)** | 身体障害者等で財産の管理が困難な方への日常的金銭管理、書類預かり等の支援。 |
| **[福祉法律相談](welfare-legal-consultation.md)** | 福祉サービスに関するトラブル・疑問や、障害者の日常生活上の法律問題についての弁護士による個別相談（要予約）。 |
| **[成年後見制度利用相談・利用支援](guardianship-consultation.md)** | [成年後見制度](adult-guardianship-system.md)や財産管理の利用についての弁護士・司法書士相談（要予約）、利用支援、および学習会の開催。 |
| **[成年後見制度申立費用助成](guardianship-application-subsidy.md)** | 低所得等の理由で[法定後見制度](statutory-guardianship.md)の申立費用が支払えない方を対象とした費用助成（※報酬の助成は区の[成年後見制度利用助成事業](guardianship-allowance.md)が担当します）。 |
| **福祉サービスに対する苦情申し立てへの対応** | 福祉サービスに対する苦情や疑問などに対し、サービス提供事業者等と共に解決に向けて対応します。 |
