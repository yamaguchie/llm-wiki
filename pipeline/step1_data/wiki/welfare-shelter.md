# 福祉避難所の設置

福祉避難所は、震災等の大規模災害が発生した際、区立小中学校等に設置される通常の避難所での生活に支障が生じる、特別な配慮が必要な方（高齢者や障害者など）を対象に二次的に開設される避難所です。

## 福祉避難所への避難の流れ

福祉避難所は二次的避難所となるため、原則としてまずは区立小中学校等の一般避難所へ避難していただく必要があります。

```
自宅が倒壊・焼失等（またはそのおそれ）
↓
避難所（区立小中学校等の体育館等）へ避難
↓
避難生活が困難な場合、要配慮者専用スペースへ移動
↓
福祉避難所（二次避難所）へ移動（必要性が高いと決定された場合）
```

※福祉避難所は、平常時は入所・通所施設等として運営されているため、災害発生後は入所者や利用者の安全確保、施設被災状況、職員配置等の確認を完了し、準備が整い次第の開設となります。

## 対象者

要介護高齢者や障害のある方のうち、一般避難所での避難生活が困難であると区が決定した方、およびその家族等の介助者（1名程度）。本人の障害状況や心身の健康状態等に基づき、必要性が高い重度の方などから優先して対象となります。

## 主な開設予定場所

* 福祉作業所（大塚、小石川）
* 障害者支援施設リアン文京
* 本郷福祉センター若駒の里、ふる里学舎本郷
* 東京都立文京盲学校
* 特別養護老人ホーム（7か所）、ショートステイ（1か所）
* 高齢者在宅サービスセンター（4か所）、介護老人保健施設（2か所）
* 認知症高齢者グループホーム（2か所）、有料老人ホーム（3か所）
