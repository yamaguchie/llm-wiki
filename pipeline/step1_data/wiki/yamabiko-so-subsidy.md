# 「四季の郷薬師温泉やまびこ荘」の利用

区内在住者が「四季の郷薬師温泉やまびこ荘」に宿泊する際に、宿泊利用補助金が支給される制度です。障害者および随伴者には補助金額が増額されます。

## 対象
区内在住で、以下の手帳をお持ちの方。
1. [身体障害者手帳](shintai-shougaisha-techou.md)
2. [愛の手帳](ai-no-techou.md)
3. [精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md)
4. 戦傷病者手帳
5. 被爆者健康手帳

### 随伴者
区内在住で、次のいずれかに該当する方に随伴する方（随伴を要する障害者1名につき1名まで）。
* [身体障害者手帳](shintai-shougaisha-techou.md) 第1種
* [愛の手帳](ai-no-techou.md) 1度〜3度
* [精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md) 1級・2級
* 視覚・聴覚及び体幹機能障害（特別項症から第4項症まで）と上肢・下肢機能障害（特別項症から第3項症まで）による戦傷病者手帳

## 助成適用後の利用料金（障害者・随伴者1人あたりの1泊2食付料金）
* **大人（中学生以上）**：4,000円（利用料金9,500円のうち5,500円を補助）※入湯税別途
* **子ども（小学生）**：3,150円（利用料金6,400円のうち3,250円を補助）
  * ※1名利用の場合は500円増
  * ※土曜日・休前日は1,000円増
  * ※12月31日から1月3日宿泊分は2,000円増

## 手続き
現地精算時に、区内在住を証明できる本人確認書類、および障害のある方は手帳（コピー可）を提示してください。
