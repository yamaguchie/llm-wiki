# ヘルプマーク

ヘルプマークは、義足や人工関節を使用している方、内部障害や難病の方、精神障害や知的障害のある方、または妊娠初期の方など、外見からは分からなくても援助や配慮を必要としている方々が、周囲に対して配慮や支援を促すために作られたマークです。

```
【ヘルプマークの視覚イメージ】
赤地に白の「プラス」マークと「ハート」マークを組み合わせたデザイン。
```

## 概要と目的
ヘルプマークをカバン等に着用することにより、周囲の人に「手助けを必要としている」ことを知らせ、電車やバス内での席の譲り合い、駅のホームや街中での安全な誘導・配慮を得られやすくすることを目指しています。

## ヘルプカードとの違い
- **ヘルプマーク**：カバン等に取り付けることで、周囲へ視覚的に「配慮が必要であること」を知らせる役割を持ちます。
- **[ヘルプカード](help-card.md)**：障害情報や緊急連絡先、配慮してほしい具体的な内容を記入して財布やパスケースに携帯するカードであり、災害や体調不良時に取り出して周囲に提示することで、的確な支援を受けるために使用されます。

## 配付対象者
- [身体障害者手帳](shantai-shogaisha-techo.md)、[愛の手帳](ai-no-techo.md)、[精神障害者保健福祉手帳](seishin-shogaisha-techo.md)の交付を受けている方
- 難病患者、妊産婦など、日常生活や災害時において援助・配慮が必要な方

## 窓口・配付場所
ヘルプマークの配付は、文京区の[障害福祉課](shogai-fukushi-ka.md)や保健福祉センター等の窓口にて無料で実施しています。また、東京都が作成した意思疎通条例（[文京区手話言語条例等](bunkyo-sign-language-ordinance.md)）等の考えに基づき、誰にでも分かりやすいカラーユニバーサルデザインとしての配慮が進められています。
