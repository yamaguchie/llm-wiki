# 郵便等投票

身体に重度の障害がある方が、自宅などで郵便等により投票できる不在者投票制度です。

## 対象となる方
原則として自書可能な方で、身体障害者手帳、戦傷病者手帳、または介護保険被保険者証を所持し、以下の要件に該当する方が対象です。事前の登録手続きが必要です。

| 障害等の区分 | 障害等 | 等級等 |
| :--- | :--- | :--- |
| 身体障害者手帳 | 両下肢、体幹、移動機能 | 1級又は2級 |
| | 心臓、じん臓、呼吸器、ぼうこう、直腸、小腸 | 1級又は3級 |
| | 免疫、肝臓 | 1級から3級まで |
| 戦傷病者手帳 | 両下肢、体幹 | 特別項症から第2項症まで |
| | 心臓、じん臓、呼吸器、ぼうこう、直腸、小腸、肝臓 | 特別項症から第3項症まで |
| 介護保険被保険者証 | 要介護状態区分 | 要介護5 |

## 郵便等投票における代理記載制度
郵便等投票の対象者のうち、自書が困難で以下の要件に該当する方は、あらかじめ届け出た代理人（選挙権を有する方）に投票の記載をさせることができます。

| 障害等の区分 | 障害等 | 等級等 |
| :--- | :--- | :--- |
| 身体障害者手帳 | 上肢又は視覚 | 1級 |
| 戦傷病者手帳 | | 特別項症から第2項症まで |

## 利用の手続き
* 投票用紙等は、投票日の4日前まで（必着）に区選挙管理委員会へ請求する必要があります。
* 投票所での投票方法として、ほかに[点字投票](braille-voting.md)や[代理投票](proxy-voting.md)があります。
