# 心身障害児総合医療療育センター

心身に障害のある子どもたちのための総合的な医療療育支援機関です。外来による療育のほか、各種福祉法に基づく入所施設や短期入所事業、療養介護事業等を備えています。

## 提供内容・部門

| 部門 | 内容 |
| :--- | :--- |
| 外来 | ・一般医療機関では対応困難な障害児・者の診療<br>・就学前の障害児と家族を対象とした児童発達支援事業による生活支援<br>・学齢の障害児と家族を対象とした放課後等デイサービス事業による生活支援 |
| 整肢療護園<br>（医療型障害児入所施設・療育介護施設） | ・肢体不自由児を対象とした治療、看護（手術・親子入園を含む）<br>・自立に必要な訓練、生活指導<br>・家族の病気や休息等で短期間のお預かりを行う短期入所事業による在宅支援 |
| むらさき愛育園<br>（医療型障害児入所施設・療育介護施設） | 重症心身障害児・者の治療、看護および日常生活の支援 |
| 歯科 | 一般の歯科医院では対応困難な障害児・者を対象とする歯科治療等 |

## 診療体制
- **診療科目**: 整形外科、小児科、精神科、リハビリテーション科、歯科
- **予約・診察時間**:
  - 【月〜金曜日】 午前9時〜午後5時
  - 【土曜日（第2、第4）】 午前9時〜午後1時
  - ※受診を希望される場合は、事前に外来に電話にてご相談ください。
