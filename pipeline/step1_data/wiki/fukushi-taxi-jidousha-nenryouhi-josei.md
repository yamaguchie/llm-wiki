# 福祉タクシー・自動車燃料費助成

移動が困難な重度障害者等の外出を支援するため、福祉タクシー券の交付や、自動車燃料費の一部助成を行う制度です。

## 申請窓口

お持ちの手帳や障害種別により、申請窓口が異なります。

- **身体障害者手帳・愛の手帳を所持している方**:
  - [障害福祉課](shougai-fukushi-ka.md) 障害者在宅サービス係（文京シビックセンター9階北側 / TEL: 03-5803-1212 / FAX: 03-5803-1352）
- **精神障害者保健福祉手帳を所持している方**:
  - 予防対策課精神保健担当（文京シビックセンター8階南側 / TEL: 03-5803-1230 / FAX: 03-5803-1355）
