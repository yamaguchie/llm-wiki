# 心身障害者・児紙おむつ支給事業

在宅で生活する重度の障害のある方（児）を対象に、排泄ケアおよび家族の介護負担軽減を支援するため、紙おむつを支給する事業です。

## 概要
失禁状態などにあり、日常生活で常時紙おむつの使用を必要とする在宅の障害者（児）に対し、排泄支援として紙おむつの支給または購入費用の一部助成を行います。
※「[心身障害者・児紙おむつ支給事業 (p.76)](diaper-distribution-service.md)」と同趣旨の制度です。

## 対象（障害程度別対象一覧）
*   **[身体障害者手帳](shintai-shougaisha-techou.md)の所持者**
    *   視覚障害：1級〜6級
    *   聴覚又は平衡感覚機能障害：2級〜6級（5・6級を含む）
    *   音声・言語機能障害：3級〜4級
    *   肢体不自由：1級〜6級
    *   内部障害：1級〜4級
*   **[愛の手帳](ai-no-techou.md)（知的障害）の所持者**：1度〜4度
