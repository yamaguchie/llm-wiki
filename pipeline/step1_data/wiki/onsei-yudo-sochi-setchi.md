# 音声誘導装置の設置

視覚障害のある方が建物の出入口等の場所を把握できるよう、音声で案内する誘導装置を公共施設等に設置しています。

## 対象
- 視覚障害者で、日常生活用具の給付制度より「歩行時間延長信号機用小型送信機」の給付を受けている方

## 設置場所

| 施設名 | 設置場所詳細 |
| --- | --- |
| **文京シビックセンター** | ・1階 出入口（礫川公園側・春日通り側）<br>・地下2階 出入口（後楽園駅側・春日駅側）<br>・3階 障害者会館出入口 |
| **真砂中央図書館** | ・1階 出入口 |
| **文京総合福祉センター** | ・1階 正面出入口<br>・3階 服部坂方面出入口<br>・4階 エレベーターホール |
| **区民センター** | ・1階 出入口 |
