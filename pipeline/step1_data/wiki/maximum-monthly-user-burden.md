# 利用者負担上限月額

障害福祉サービス及び障害児支援の利用者負担は、所得（世帯の収入状況）に応じて負担上限月額が設定されています。ひと月にどれだけサービスを利用しても、この上限額以上の負担は生じません。

## 負担上限月額の一覧

| 区分 | 世帯の収入状況 | 負担上限月額 |
| :--- | :--- | :--- |
| **生活保護** | 生活保護受給世帯 | 0円 |
| **低所得1** | 区市町村民税非課税世帯のうち、本人の年収80万円以下 | 0円 |
| **低所得2** | 区市町村民税非課税世帯（低所得1に該当する方を除く） | 0円 |
| **一般1** | 区市町村民税課税世帯で所得割16万円未満の方（※障害児及び20歳未満の施設入所者の場合は28万円未満） | 居宅で生活する障害児：**4,600円**<br>居宅で生活する障害者及び20歳未満の施設入所者：**9,300円** |
| **一般2** | 区市町村民税課税世帯（一般1に該当する方を除く） | **37,200円** |

*   ※補装具の負担上限月額は、課税世帯の場合「一般2」のみとなります。また、一定所得以上の世帯の方は対象外となります。
*   ※入所施設利用者（20歳以上）、グループホーム利用者の方で課税世帯の場合は「一般2」となります。

## 所得を判断する世帯の範囲

| 種別 | 世帯の範囲 |
| :--- | :--- |
| **18歳以上の障害者**（施設に入所する18・19歳を除く） | 障害のある方と配偶者 |
| **障害児**（施設に入所する18・19歳を含む） | 保護者の属する住民基本台帳での世帯 |

## 主な軽減制度
*   **[幼児教育・保育無償化](free-early-childhood-education-and-care.md)**（3〜5歳の通所支援等無償化）
*   **[就学前の障害児通所支援に係る利用者負担額等の助成](bunkyo-pre-school-child-support-grant.md)**（文京区独自の0〜2歳児等の助成）
*   **[同行援護及び移動支援の利用者負担減免](reduction-of-user-burden-for-guide-and-movement-support.md)**（月36時間まで無料）

*   **食費等実費負担の軽減**（通所・入所施設、グループホームの食費・光熱水費・家賃負担軽減）
