# 日常生活用具の給付

在宅の重度障害者等の日常生活を円滑にするため、日常生活用具の購入・レンタルを給付または助成する制度です。

## 申請窓口

対象となる方の障害種別等により、窓口が異なります。

- **身体障害のある方・児童**:
  - [障害福祉課](shougai-fukushi-ka.md) 身体障害者支援係（文京シビックセンター9階北側 / TEL: 03-5803-1219 / FAX: 03-5803-1352）
- **精神障害（発達障害含む）のある方・難病患者、小児慢性特定疾病児童**:
  - 予防対策課精神保健担当（文京シビックセンター8階南側 / TEL: 03-5803-1847 / FAX: 03-5803-1355）

※難病（特殊疾病）や小児慢性特定疾病医療費の助成等についての健康相談は、保健サービスセンターまたは保健サービスセンター本郷支所（TEL: 03-3821-5106）でも受け付けています。
