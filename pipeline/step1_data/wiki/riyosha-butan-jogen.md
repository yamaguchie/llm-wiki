# 利用者負担上限月額

**[自立支援給付](jiritsu-shien-kyufu.md)**やその他の障害福祉サービスを利用する場合、利用者は原則としてサービスにかかる費用の1割を支払いますが、世帯の所得状況等に応じてひと月に支払う「利用者負担上限月額」が設定されており、それを超える負担は発生しません。

## 所得区分と上限月額の目安

利用者の世帯所得区分（18歳未満の障害児の場合は、保護者の属する世帯）に基づき、以下のように上限月額が設定されます。

| 区分 | 世帯の所得状況 | 負担上限月額 |
| :--- | :--- | :--- |
| **生活保護** | 生活保護受給世帯 | **0円** |
| **低所得** | 市民税（特別区民税）非課税世帯 | **0円** |
| **一般1** | 市民税（特別区民税）課税世帯（所得割16万円未満）※障害児通所支援等の場合 | **4,600円** |
| **一般2** | 上記以外の市民税（特別区民税）課税世帯 | **37,200円** |

※具体的な世帯の範囲や資産要件により異なる場合があります。

## 特例や更なる軽減制度

- **[幼児教育・保育無償化](yoji-kyoiku-hoiku-mushoka.md)**：3歳から5歳児を対象とする障害児通所支援などの利用者負担額は無償化（0円）されます。
- **[文京区の負担軽減制度](bunkyo-ku-futan-keigen.md)**：就学前児童の利用料助成や、特定の福祉サービスに関する利用者負担の免除など、文京区が独自に行う上乗せの軽減措置が用意されています。
