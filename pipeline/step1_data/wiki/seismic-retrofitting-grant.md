# 耐震改修工事等の助成

昭和56年5月31日以前に着工された住宅、または昭和56年6月1日から平成12年5月31日までの間に着工された木造住宅（在来軸組工法による平屋または2階建てに限る）の所有者が、耐震診断や耐震改修工事等を行うための費用の一部を区が助成する制度です。

高齢者または障害者が居住する木造住宅は、助成の補助率と補助金上限額が優遇されます。契約前に申請の手続きが必要です。

## 対象者
住宅の所有者であり、次のいずれかに該当する方。
1. 本人または3親等以内の親族が、申請時に満65歳以上で、助成対象住宅に1年以上居住している。
2. 本人または3親等以内の親族が、申請時に[身体障害者手帳](shintai-shougaisha-techou.md)1〜4級、もしくは[愛の手帳](ai-no-techou.md)1〜3度に該当し、助成対象住宅に1年以上居住している。

## 助成内容

| 助成種類 | 助成金額 | 対象地区 |
| :--- | :--- | :--- |
| **木造建築物耐震診断助成** | 耐震診断に要した費用の10/10（上限20万円） | 区内全域 |
| **木造住宅耐震改修工事助成** | 助成対象設計費・工事費の3/4（上限240万円）※すでに耐震設計助成を受けている場合は上限200万円 | 準防火地域内 |
| **木造住宅耐震シェルター等助成** | 助成対象工事費の3/4（上限40万円） | 区内全域 |
