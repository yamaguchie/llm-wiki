# 障害者・児移動支援事業

屋外での移動に著しい制限がある障害者（児）に対し、社会生活上必要な外出や余暇活動などの社会参加に伴う外出の介護支援を行います。

## 対象者
1. 身体障害者・児
2. 知的障害者・児
3. 精神障害者・児
4. 難病等患者・児（対象疾患は元文書199ページを参照）

## 費用
利用者負担は、世帯の収入・所得に応じて設定されます（[利用者負担上限月額](maximum-monthly-user-burden.md)）。
ただし、区独自の減免制度により、**月36時間までは利用者負担額はかかりません**（詳細は[同行援護及び移動支援の利用者負担減免](reduction-of-user-burden-for-guide-and-movement-support.md)を参照）。

## 特別な外出支援
- **通学支援**: 利用者本人および家族の状況を考慮し、1回あたり1時間まで、月10回を限度として利用できます。ただし、保護者の疾病等の個別事情が認められる場合は、月23回まで増やすことができます。
- **通所支援**: 生活介護施設への通所で送迎サービスを利用できない場合や、地域活動支援センターへの送迎に利用できます。生活介護以外の施設であっても、通所に慣れる目的として一定期間に限り利用可能な場合があります。
