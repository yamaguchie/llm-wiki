# 中途失聴者・難聴者手話講習会

中途失聴者および難聴者の方を対象に、日常生活で役立つ手話技術の習得を目的として開催される講習会です。

## 概要
簡単な意思交流から、日常会話が可能な程度の手話技術を学ぶ講習会です。

## 対象
* 東京都在住・在勤の中途失聴者・難聴者

## 受講詳細
* **費用**：無料（ただし、教材費は自己負担）
* **会場**：
  * 三田会場（東京都障害者福祉会館）
  * 多摩会場（[東京都障害者スポーツセンター](tokyo-disabled-sports-center.md) / 東京都多摩障害者スポーツセンター等）
  * ※開催日程等については、東京都福祉局のホームページを確認するか、直接お問い合わせください。
