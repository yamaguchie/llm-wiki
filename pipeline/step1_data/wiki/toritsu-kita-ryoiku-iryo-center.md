# 都立北療育医療センター

児童福祉法に基づく医療型障害児入所施設、児童発達支援センター、および障害者総合支援法に基づく療養介護、生活介護を提供するほか、一般医療機関では対応が困難な心身障害児・者に対する一般疾病の診療を行う医療施設です。

## 提供内容・部門

| 部門 | 内容 |
| :--- | :--- |
| 入院、外来 | 一般医療機関では対応困難な障害児・者の診療 |
| 医療型障害児入所施設・児童発達支援センター | 肢体不自由児を対象に医療・看護ケアを提供するとともに、自立に必要な支援を行う |
| 療養介護 | 重症心身障害児・者を保護するとともに、医療・看護ケアの提供および日常生活の支援を行う |
| 生活介護（旧重症心身障害者通所） | 重症心身障害者に当センターへ通っていただき、医療・看護上の支援および日常生活の支援を行う |
| 障害者歯科 | 一般の歯科医院では対応困難な障害児・者を対象とする歯科診療 |

## 診療科目
内科、脳神経内科、精神科、小児科、外科、整形外科、リハビリテーション科、皮膚科、泌尿器科、眼科、耳鼻咽喉科、歯科

## 診療日
- 月〜金曜日（※受診には予約が必要です）
