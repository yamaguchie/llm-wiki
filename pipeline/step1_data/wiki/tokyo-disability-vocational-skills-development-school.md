# 東京障害者職業能力開発校

就職に向けて必要な職業上の知識や技能を学び、ハローワークと連携して就職の相談・支援を行う施設です。

- **対象区分**：[身体障害者手帳](shintai-shougaisha-techou.md)（身）、[愛の手帳](ai-no-techou.md)（知）、[精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md)（精）

## 対象
1. 一般向けの職業能力開発センターで職業訓練を受けることが困難な身体障害者、知的障害者、精神障害者、および発達障害者
2. ハローワークに求職登録し、障害や症状が安定して職業訓練を継続して受講できる方
   ※身体障害者（重度視覚障害者を除く）は、一定条件を満たす場合、選考の上で入寮が可能です。

## 訓練科目
| 対象 | 科目 | 期間 |
| :--- | :--- | :--- |
| 身体障害者、精神障害者、発達障害者 | 調理・清掃サービス科 / オフィスワーク科 | 6か月 |
| 身体障害者、精神障害者、発達障害者 | ビジネスアプリ開発科 / ビジネス総合事務科 / グラフィックDTP科 / ものづくり技術科 / 建築CAD科 / 製パン科 | 1年間 |
| 精神障害者、発達障害者（※医師の診断書・手帳所持者等） | 職域開発科 | 6か月 |
| 身体障害者、精神障害者、発達障害者 | 就業支援科 | 3か月 |
| 知的障害者（愛の手帳所持者等） | 実務作業科 | 1年間 |
| 重度視覚障害者 | OA実務科（※委託先で実施） | 1年間 |

## 費用
- **受講料**：無料（作業代などは自己負担となります）
- 公共職業安定所長の指示により、訓練手当が支給される場合があります。

## 募集と申込
- **募集時期**：毎年9月以降（6か月および3か月訓練科目の場合は、4月、7月、10月、1月の年4回募集）
- **申込先**：[飯田橋公共職業安定所（ハローワーク飯田橋）](hellowork-iidabashi.md) 専門援助第二部門
