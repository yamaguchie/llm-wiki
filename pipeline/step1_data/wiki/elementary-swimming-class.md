# 初級水泳教室

初級水泳教室は、障害のある方を対象に、水への恐怖心を和らげることや、基本的な水泳技術を身につけることを目的として実施される教室です。

## 概要
専門のインストラクターが個別のレベルや障害の状況に合わせて丁寧に指導を行います。泳ぎの基本動作を学ぶだけでなく、体力づくりや健康増進を目的としています。

## 受講対象
- 区内在住・在勤・在学の障害児・障害者（具体的な対象年齢や手帳要件は募集要領によります）。
- [身体障害者手帳](shantai-shogaisha-techo.md)や[愛の手帳](ai-no-techo.md)をお持ちの方等。

## 関連事業
- 水中での遊びや親水活動を主目的とする[プールレクリエーション](pool-recreation.md)も開催されています。
- 施設利用費用の減免については[区立施設等の使用料・利用料金の免除・割引](ward-facility-fee-waiver.md)が適用される場合があります。
- 問い合わせ先：[障害福祉課](shogai-fukushi-ka.md)
