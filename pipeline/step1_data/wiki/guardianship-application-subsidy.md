# 成年後見制度申立費用助成

成年後見制度申立費用助成は、[権利擁護センター「あんしんサポート文京」](anshin-support-bunkyo.md)が実施する、経済的な理由等で法定後見制度の利用申立てが困難な方に向けた費用助成事業です。

## 助成対象
低所得などの理由により、[法定後見制度](statutory-guardianship.md)の申立費用（家庭裁判所への申立て時に必要となる実費等）を支払うことが困難な方。

## 注意事項
- 本事業は「申立費用」のみを対象としています。
- 成年後見人、保佐人、補助人への「報酬」に関する助成については、文京区（福祉政策課）が実施する[成年後見制度利用助成事業](guardianship-allowance.md)の対象となります。
