# ハローワーク飯田橋

ハローワーク飯田橋（飯田橋公共職業安定所）は、千代田区、中央区、文京区などを管轄する公共職業安定所です。障害をお持ちの方を対象とした「専門援助部門（障害者相談窓口）」を設置し、一貫した求職支援を行っています。

## 障害者向けサービス内容

- **職業紹介・求職登録**:
  障害者雇用枠および一般枠を含めた求人案件を提示し、求職者の特性に応じた紹介状を発行します。
- **就職支援プログラム**:
  面接対策や履歴書の書き方指導、就職に向けた個別カウンセリングを実施します。
- **企業へのアプローチ**:
  障害者雇用を検討または実施している企業に対し、採用サポートや各種助成金制度等の案内を行います。

## 近隣機関との共同支援
- 地域における定着支援や、きめ細かいカウンセリングを並行して行うため、[文京区障害者就労支援センター](bunkyo-disability-employment-support-center.md)と緊密な情報共有を行っています。
- 職業的な適性検査や専門リハビリが必要な場合は、[東京障害者職業センター](tokyo-disability-vocational-center.md)を紹介・仲介します。
