# 愛の手帳

知的障害のある方が、各種のサービスを受けるために、東京都が独自に発行している手帳（療育手帳）です。障害の程度により、1〜4度に区分されています。

## 新規申請・年齢更新

愛の手帳に有効期限はありませんが、**3歳・6歳・12歳・18歳**に達した際には、年齢更新の再判定を受けてください。他の道府県の療育手帳をお持ちの方が文京区に転入した場合は、新たに愛の手帳を新規申請し、判定を受ける必要があります。

事前に以下の申請先へ電話予約をした上で、判定を受けてください。

### 申請・判定先
- **18歳未満の方**: 文京区児童相談所（文京区小石川3-14-7 / TEL: 03-3811-5241）
- **18歳以上の方**: 東京都心身障害者福祉センター（新宿区神楽河岸1-1 セントラルプラザ14階 / TEL: 03-3235-2961）

## 内容変更・返還・再交付手続き

手帳の交付を受けている方で、以下の事由が発生した場合は、[障害福祉課](shougai-fukushi-ka.md)（知的障害者支援係）で手続きを行ってください。

| 区分 | 持ち物 |
| --- | --- |
| 居住地の変更・氏名の変更・保護者の変更（氏名、居住地）・本人の死亡 | 手帳 |
| 手帳を紛失、破損又は汚損したとき | 破損・汚損した手帳、顔写真（脱帽・上半身、縦4cm×横3cm） |
| カード形式の手帳へ変更するとき | 手帳、顔写真（脱帽・上半身、縦4cm×横3cm） |
