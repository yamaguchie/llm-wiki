# 磁気ループの設置

磁気ループの設置は、難聴者や補聴器を利用している方が、窓口やホールなどの騒音下でもクリアに音声を聞き取れるようにするための音響支援です。

## 概要
区の施設の窓口やイベントホール等に、補聴器誘導システム（磁気ループ等）を設置しています。このシステムを稼働させることで、周囲の雑音に影響されず、必要な音声を補聴器等へ直接届けることができます。
