# 自立支援医療

心身の障害を軽減・除去するための医療について、医療費の自己負担額を軽減する公費負担医療制度です。「更生医療」「育成医療」「精神通院医療」の3つの区分があります。

## 自立支援医療の種類と窓口

### 1. 自立支援医療（更生医療）
18歳以上の身体障害者手帳所持者が、障害を除去・軽減するための温熱療法や手術等の指定医療を受ける場合の助成です。
- **担当窓口**: [障害福祉課](shougai-fukushi-ka.md) 身体障害者支援係（文京シビックセンター9階北側 / TEL: 03-5803-1219）

### 2. 自立支援医療（育成医療）
身体に障害がある18歳未満の児童に対し、生活能力向上に必要な指定医療機関での治療費を助成する制度です。
- **担当窓口**: 健康推進課健康増進係（文京シビックセンター8階南側 / TEL: 03-5803-1961 / FAX: 03-5803-1355）

### 3. 自立支援医療（精神通院医療）
精神疾患のある方の通院治療に必要な医療費の自己負担を原則1割に軽減する制度です。
- **担当窓口**: 
  - 予防対策課精神保健担当（文京シビックセンター8階南側 / TEL: 03-5803-1230 / FAX: 03-5803-1355）
  - 保健サービスセンター 本郷支所（TEL: 03-3821-5106 / FAX: 03-3822-9174）
