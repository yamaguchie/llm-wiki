# マイファイル「ふみのわ」

**マイファイル「ふみのわ」**は、文京区が独自に作成・配付している、障害のある子どもや発達が気になる子どものための「成長記録・支援連携ファイル」です。

## 目的
乳幼児期から学校卒業、さらには成人期・社会への移行期に至るまで、子どもの発達状況、受けてきた支援内容、医療情報、特性や配慮事項などを一元的にまとめて保管・記録することができます。
保護者がこれを持ち、進学（園）や相談機関の変更、新たなサービスを利用する際に提示することで、関係者へ正確な情報をスムーズに伝え、一貫した支援を受けることが可能になります。

## 主な特徴と活用場面
- **成長のバトンパス**：保育園（[保育園の要配慮児保育](hoikuen-yohaioruji-hoiku.md)）から小学校、中学校、特別支援教育（[特別支援教育相談](tokubetsu-shien-kyoiku-sodan.md)）への進級時に、学校側へ個別の配慮事項を説明する際に活用。
- **相談の効率化**：[教育センター 総合相談室](kyoiku-center-sogo-sodanshitsu.md)や[文京区障害者基幹相談支援センター](bunkyo-kikan-sodan-center.md)などでの相談時に、これまでの成育歴を説明する手間を省けます。
- **緊急時の備え**：[ヘルプカード](help-card.md)や緊急連絡先とともに、医療情報（アレルギー、服用薬）を整理しておくことで、災害時にも役立ちます。
