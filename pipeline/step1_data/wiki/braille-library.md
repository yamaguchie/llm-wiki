# 点字図書館

点字図書館は、視覚障害のある方の情報保障と読書環境を整備するため、点字図書や録音図書の製作・貸出、日常生活を支える相談等を行う専門施設です。

## 概要
主な活動内容および提供サービスは以下の通りです。
* 点字図書や録音図書（デイジー録音など）の製作および貸出サービス
* 視覚障害のある方に向けた対面朗読サービス
* 視覚障害者向けの福祉用具や生活便利グッズの開発、展示、普及活動

## 関連する支援・サービス
* [点字図書の給付](braille-book-provision.md)：購入にかかる差額を区が負担する給付制度です。
* [点字・声の広報](braille-voice-public-relations.md)：区報や区議会だより等の点字版・音声版を届けるサービスです。
* [音声誘導装置の設置](acoustic-guidance-system-installation.md)：施設出入口等で案内音声を流す装置です。
* [区立図書館の利用支援](municipal-library-support.md)：区立図書館で行っている宅配や対面朗読等の支援事業です。
