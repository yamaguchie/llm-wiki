# 避難行動要支援者名簿

災害時または災害が発生するおそれがある場合、自力で避難することが困難で支援を必要とする方を事前登録し、避難支援を円滑にするための名簿制度です。災害対策基本法に基づき作成されています。

## 名簿の運用方式

| 名簿の種類 | 登録対象者 | 平常時の情報管理 | 災害時の情報共有範囲 |
| :--- | :--- | :--- | :--- |
| **関係機関共有方式名簿** | 区が指定する避難行動要支援者すべて、およびそれ以外で登録を希望する方 | 区、高齢者あんしん相談センター、[文京区障害者基幹相談支援センター](bunkyo-shougaisha-kikan-soudan-shien-center.md) | 左記の機関に加え、区民防災組織（町会・自治会）、民生委員・児童委員、消防署、警察署、文京区社会福祉協議会、福祉サービス事業者 |
| **同意方式名簿** | 関係機関共有方式名簿の登録者のうち、平常時からの情報共有に同意した方のみ | 区、高齢者あんしん相談センター、[文京区障害者基幹相談支援センター](bunkyo-shougaisha-kikan-soudan-shien-center.md)、および上記避難支援者等関係者 | 同上 |

## 登録対象者

### 1. 自動的に登録される方（区が指定する避難行動要支援者）
以下のいずれかの条件に該当する方は、関係機関共有方式名簿に自動で登録されます。平常時からの共有を希望する場合は「同意書」の提出が必要です。
* 要介護3〜5の認定を受けている方
* [身体障害者手帳](shintai-shougaisha-techou.md)の特定等級の方（上肢1〜2級、下肢1〜2級、体幹1〜3級、視覚1〜2級、聴覚2級）
* [愛の手帳](ai-no-techou.md) 1〜3度の方
* [精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md) 1級の方
* 難病医療費を受給しており、日常生活が全介助の方

### 2. 希望により同意書を提出して登録する方
上記に該当しない方でも、以下のいずれかに該当し、かつ平常時から区民防災組織等へ情報共有することに同意できる場合は、申請により登録が可能です。
* 65歳以上の単身世帯、または65歳以上の高齢者のみの世帯
* 要介護または要支援の認定を受けている方
* [身体障害者手帳](shintai-shougaisha-techou.md)をお持ちの方
* [愛の手帳](ai-no-techou.md)をお持ちの方
* [精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md)をお持ちの方
* 難病医療費受給者
