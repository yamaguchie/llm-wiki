# 重度障害者特別給付金（区制度）

重度障害者特別給付金は、障害基礎年金等の受給資格を有しない重度の障害をお持ちの在日外国人で、特定の基準を満たしている方に支給される文京区独自の給付金制度です。

## 対象者

以下の条件のすべてに該当する特別永住者の方が対象となります。

- 重度障害（[身体障害者手帳](shintai-shougaisha-techou.md)1・2級、[愛の手帳](ai-no-techou.md)1・2度、または[精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md)1・2級）をお持ちの方
- 日本国籍を有していなかったため国民年金に加入することができず、[国民年金（障害基礎年金）](disability-basic-pension.md)等を受けられなかった方
- 次のいずれにも該当する方：
  1. 昭和57年1月1日より前に20歳に達している方
  2. 昭和57年1月1日より前に重度障害者であった方、あるいは同日以降に重度障害者となり、その障害の発生原因となる初診日が20歳以後でかつ同日前である方

## 受給できない方

以下のいずれかに該当する方は受給できません。

1. 公的年金受給者
2. 生活保護受給者
3. 文京区に住民登録をした日から2年を経過していない方
4. 本人・配偶者又は扶養義務者の所得が限度額を超える方

## 給付額

- **手当額**: 月額 30,000円

## 必要書類

申請には以下の書類等が必要です。

1. [身体障害者手帳](shintai-shougaisha-techou.md)、[愛の手帳](ai-no-techou.md)又は[精神障害者保健福祉手帳](seishin-shougaisha-hoken-fukushi-techou.md)
2. 対象条件の「2」に合致することを証明できる診断書等
3. 印鑑
4. 預金通帳又はキャッシュカード（※本人名義のもの）
